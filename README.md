#  Warehouse Stock Management System (PHP)

Sistem ini merupakan aplikasi **pengelola stok gudang** berbasis **PHP** yang digunakan untuk mencatat, memantau, dan mengelola data persediaan barang secara efisien. Aplikasi ini dirancang sederhana namun fungsional, sehingga cocok digunakan untuk kebutuhan gudang kecil hingga menengah.

---

##  Fitur Utama

- **Manajemen Data Barang**  
  Tambah, ubah, hapus, dan lihat detail barang.

- **Kategori & Pengelompokan Barang**  
  Memudahkan pengaturan dan pemfilteran barang.

- **Pencatatan Stok Masuk & Keluar**  
  Setiap pergerakan barang (in/out) dicatat untuk menjaga akurasi stok.

- **Laporan Stok**  
  Menampilkan ringkasan stok terkini sesuai perhitungan otomatis dari sistem.

- **Manajemen Pengguna (optional)**  
  Mendukung pengaturan hak akses dan akun admin.

---

##  Teknologi yang Digunakan

- PHP Native
- MySQL / MariaDB
- HTML, CSS, JavaScript
- Apache (XAMPP / Laragon / LAMPP)
- Git & GitHub untuk version control
